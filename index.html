Цель Переход к оформлению доставки товара
Область действия Сайт пиццерии
Уровень Цели пользователя
Основное действующее лицо Любой пользователь сайта
Участники Отсутствуют
Предусловие Пользователь хочет оформить доставку продукта
Гарантии успеха Продукт добавлен в корзину, пользователь переходит к оформлению заказа
Триггер В любой момент, когда пользователь находится на детальной странице товара
Описание 1.Пользователь регистрируется на сайте
  • Заполняет необходимые поля;
  • Получает подтверждающий код.

2.Пользователь выбирает из доступного перечня товаров необходимый для него.

3.Указывает необходимое количество товара.

4.Система проверяет наличие выбранного товара.

5.Система подтверждает возможность заказа товара.

6.Пользователь перемещает товар в корзину.

7.Система добавляет товар (-ы) в состав корзины пользователя.
Альтернативные варианты
1.1 Пользователь уже зарегистрирован, для входа необходимо авторизоваться введя свой номер телефона и пароль.

1.2 Пользователь уже зарегистрирован и сохранил свои данные для автоматической авторизации, заходит в личный кабинет автоматически.

2.1 Пользователь выбирает из доступного перечня товаров необходимый для него:
  • Просматривает ингредиенты используемые в конкретном продукте изменение;
  • Выбирает дополнительные топинги представленные на выбор.
Нюансы и исключения
1.1 Если пользователь забыл пароль, необходимо нажать кнопку сбросить пароль, подтвердить номер телефона и ввести новый пароль.

5.1 Если товара нет в наличии, система оповещает клиента и предлагает заменить товар

7.1 Если пользователь захочет изменить топинг в заказе, необходимо нажать на позицию и выбрать другой топинг.
